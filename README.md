# 🤖 AI Data Analysis Agent

Upload any CSV file and get instant AI-powered business insights, visualizations, and recommendations.

## 🚀 Features
- Upload any CSV file and instantly preview the data
- Auto-generated charts and visualizations
- AI-generated business insights powered by Claude API
- Key findings and actionable recommendations

## 🛠️ Tech Stack
- Python, pandas, Streamlit
- Plotly (interactive charts)
- Anthropic Claude API (AI insights)

## 💡 How It Works
1. Upload a CSV file
2. Explore the data preview and visualization
3. Enter your Anthropic API key
4. Click "Generate Insights" — Claude analyzes your data and returns business recommendations

## 📌 Use Case
Designed for business analysts and data teams who want to quickly extract insights from raw data without writing code.
